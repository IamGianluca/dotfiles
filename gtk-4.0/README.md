# GTK-4 themes for Gruvbox and Rosé Pine

### Installation

This folder contains two CSS files, one for each theme: Gruvbox and Rosé Pine. To install a theme, rename the desired file to gtk.css. Rename the other file to something else (e.g., gtk.css.bak) so it isn't picked up by GTK.

### Icons

For Gruvbox, [`gruvbox-plus-icon-pack` by SylEleuth](https://github.com/SylEleuth/gruvbox-plus-icon-pack#installation) offers a great icon set. Follow the installation instructions in the repository, then use `lxappearance` to select the icon set.
